Privacy Policy - Learn Spanish Extension

Last updated: May 2025

Overview

This Privacy Policy describes how the "Learn Spanish" Chrome Extension ("we," "our," or "the extension") handles information. We are committed to protecting your privacy and being transparent about our data practices.

Data Collection

## Data Collection and Use ### Data We Collect: - **Advertising ID**: Used by Unity Ads for personalized advertising - **Email Address**: For user authentication and account management - **Game Progress**: Scores, levels completed, and gameplay statistics - **Device Information**: For app functionality and analytics ### Third-Party Services: - **Unity Ads**: Collects advertising ID for ad delivery - **Firebase**: Stores user data and provides analytics - **Google Play Services**: Provides authentication and analytics ### Data Sharing: We share data with Unity (for advertising) and Google (for Firebase services) as described in their respective privacy policies.

What We Store Locally

Learning Progress Data

  • Your selected Spanish proficiency level (A1–C2)
  • Subcategory preference (Beginner, Intermediate, Advanced)
  • Words you have learned and their mastery levels
  • Quiz scores and completion statistics
  • Learning streaks and daily progress

Gamification Data

  • Experience points (XP) and user level
  • Unlocked achievements and dates achieved
  • Daily challenge progress and completion status
  • Perfect quiz counts and other game statistics

User Preferences

  • Daily word learning goals (1–10 words per day)
  • Notification time preferences
  • Audio settings (sound effects on/off)
  • Motivational message preferences
  • Active tab preferences

Quiz and Test Data

  • In-progress quiz states (to resume if popup is closed)
  • Level test answers and progress (during assessment)
  • Question templates and user responses

What We DON'T Collect

We explicitly do NOT collect, store, or transmit:

  • Personal information (name, email, phone)
  • Browsing history or website visit data
  • Location or device information
  • Usage analytics or behavioral tracking
  • Anything that identifies you personally
  • Data to or from external servers

Data Storage

All data is stored using Chrome's chrome.storage.sync API:

  • Locally on your device
  • Syncs across Chrome browsers if using the same Google account
  • Managed by Chrome, without our server involvement

Data Sharing

We do not share any data with third parties because we don’t collect any data to share.

Third-Party Services

The extension does not integrate with or send data to any third-party services, analytics platforms, or external APIs.

Permissions Explained

Storage Permission

  • Purpose: Save progress, preferences, and stats
  • Scope: Limited to extension functionality

Alarms Permission

  • Purpose: Set daily learning reminder notifications
  • Scope: Only for scheduled reminders

Notifications Permission

  • Purpose: Send motivational learning reminders
  • Scope: Related only to your learning progress

Data Security

  • All data is stored locally
  • No transmission over networks
  • No external database vulnerabilities
  • Protected by Chrome’s built-in systems

Data Retention

Your data remains until:

  • You uninstall the extension
  • You reset data through settings
  • You manually clear Chrome extension data

To delete data:

  1. Use “Reset Progress” in Settings tab
  2. Uninstall the extension
  3. Clear extension data in Chrome settings

Children’s Privacy

No personal information is knowingly collected from children under 13, making this COPPA-compliant by design.

Changes to This Policy
